<p align="center">
  A custom simple notepad program, built in purely C#.<br>
</p><br>
<h2>Features</h2>
<br>

* **<h3>Auto-saving</h3>**
Be able to have your work auto-save, without any problems with memory/CPU usage. You also have the ability to determine the length between each autosave (out of a set of options). In the future I will may add being able to type any time desired.<br><br>
  
* **<h3>Dark Theme (Coming soon)</h3>**
This is one of my personal favourite features that I will be adding, no more having to be forced with just working with a light theme program, you can now use dark theme and have a more comfortable experience when making notes.<br><br>
  
* **<h3>Line Numbers (Coming soon)</h3>**
Line numbers will help you be able to keep track of which line you are on, so you don't suddenly get lost when you were suddenly on line 1337, and find yourself on 1500.<br><br>

<h2>Building</h3>
<br>
If you plan to use this program, I will not provide any support as it is very simple, however I will tell you, there are external libraries that you'll have to attach yourself via NuGet. You will have to get *Newtonsoft.Json* and optionally *Bunifu.UI*. If you can not get it then you'll have to replace bunifu elements manually.
